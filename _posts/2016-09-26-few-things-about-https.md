* 网景除了浏览器被人记住以外，还有SSL， CSS 以及 JS
* SSL和TLS只是同一个东西在不同时期的叫法
* HTTP 1.1 以后，默认采用长连接(Keep-Alive)， 因为短连接时耗费在建立TCP的时间成本和CPU成本太高。
